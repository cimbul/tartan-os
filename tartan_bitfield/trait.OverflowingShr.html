A type with an overflowing right shift operation. Also adds a saturating version.

All basic numeric types have this operation, but there is no corresponding trait in core::ops.

Required methods

fn overflowing_shr(self, n: u32) -> (Self, bool)

Shift the value right by n mod m bits, where m is the number of bits in the type. Return the shifted value along with a boolean indicating whether the shift count was wrapped.

Since this behavior is counterintuitive and practically useless, see saturating_shr for an alternative that behaves the way you probably expect.
